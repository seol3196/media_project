## 활동3 여론전쟁 화면 개선

기존에는 SNS, 뉴스 기사, 유튜브 등 플랫폼 유형이 텍스트로만 표시되고 실제 화면은 모두 같은 게시판 형태라 몰입감이 낮았다. 이를 해결하기 위해 활동3 학생 화면의 게시물 렌더링을 플랫폼별 UI로 분리했다.

현재 구현된 플랫폼 스타일:

- A조: SNS 피드 형태
- B조: 실시간 SNS/X 형태
- C조: 유튜브 영상 페이지 형태
- D조: 학교 커뮤니티 게시판 형태
- E조: 뉴스 기사 형태
- F조: 온라인 청원 형태

`PostCard`가 게시물 팀 정보를 기준으로 각 플랫폼 전용 컴포넌트를 선택해서 렌더링한다. 관련 헬퍼와 컴포넌트는 `Phase3.jsx` 안에 함께 정리되어 있다.

## 추가 반영 요청 사항

1. `편견가면 쪽지`로 보이던 표현을 모두 `미션 쪽지`로 변경했다.

2. SNS 게시물에는 마라탕 관련 사진을 추가했다.
   - 에셋 위치: `client/src/assets/phase3/malatang-lunch.png`
   - AI 티가 덜 나도록 단순한 급식 트레이 위 마라탕 사진 스타일로 생성했다.
   - `Phase3.jsx`에서 `malatangLunchImage`로 import해서 SNS 피드 이미지 영역에 사용한다.

3. 여론 조작 댓글 작성 영역을 게시글 아래로 이동했다.
   - 기존에는 오른쪽 사이드 영역에 댓글 입력창이 있었다.
   - 현재는 게시글을 본 뒤 바로 아래에서 `조작 댓글 작성` textarea와 `댓글 등록` 버튼을 사용할 수 있다.
   - 오른쪽 사이드는 `미션 쪽지`, 미션 내용, `준비완료` 상태만 담당한다.

4. 여론 조작 이후 댓글 달기 단계의 게시물 목록을 익명화했다.
   - 기존에는 목록에서 제목, 유형, 플랫폼 정보가 노출될 수 있었다.
   - 현재는 `게시물 1`, `게시물 2`, `게시물 3`처럼 번호만 표시한다.
   - 옆 학생이 화면을 봐도 어떤 게시물에 댓글을 달았는지 추측하기 어렵게 했다.

5. 뉴스 기사 화면에서 불필요한 쟁점 요약 박스를 삭제했다.
   - 삭제한 문구:
     - `쟁점 요약`
     - `찬성과 반대 의견이 맞서는 사안을 학생 여론으로 확인 중입니다.`
     - `댓글 반응에 따라 분위기가 빠르게 바뀔 수 있습니다.`
   - 현재 뉴스 기사는 기사 제목, 기자/메타 정보, 본문 중심으로 보인다.

## 확인한 사항

- `rg`로 `편견가면`, `쟁점 요약`, 삭제 대상 문구가 `Phase3.jsx`에 남아 있지 않은 것을 확인했다.
- `npm run build`가 정상 통과했다.
